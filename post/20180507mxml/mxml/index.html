MXML的使用: The description of mxml

MXML (Minimal XML) is a small, fast and versatile library that reads a whole XML file and puts it in a DOM tree.

1.XML

XML特点和作用 :

  • XML 指可扩展标识语言（ eXtensible Markup Language）
  • XML 的设计宗旨是传输数据， 而非显示数据
  • XML 标签没有被预定义。 您需要自行定义标签。
  • 作为一种通用的数据存储和通信格式被广泛应用。
  • 描述的数据作为一棵树型的结构而存在。
1<?xml version="1.0" encoding="utf-8"?>
2<books>
3    <book id="001">
4        <author>wanger</author>
5    </book>
6</books>

2.mxml写入操作

mxml既可以创建写入xml文件，也可以解析xml文件数据
写入示例:

 1#include"mxml.h"
 2int main()
 3{
 4    // create a new xml
 5    mxml_node_t *xml = mxmlNewXML("1.0");
 6    //add a node name books
 7    mxml_node_t *books = mxmlNewElement(xml,"books");
 8    //add a node under books name book
 9    mxml_node_t *book = mxmlNewElement(books,"book");
10    //set attr
11    mxmlElementSetAttr(book,"id","001");
12    mxml_node_t *author = mxmlNewElement(book,"author");
13    mxmlNewText(author,0,"wanger");
14
15    FILE* fp = fopen("book.xml","wb");
16    mxmlSaveFile(xml,fp,MXML_NO_CALLBACK);
17    fclose(fp);
18    mxmlDelete(xml);
19    return 0;
20}

3.mxml解析操作

解析步骤：
步骤：

  1. 打开一个xml文件
  2. 把文件加载到内存中，mxml_node_t mxmlLoadFile(mxml_node_t *top, FILE *fp,mxml_type_t (*cb)(mxml_node_t ));
  3. 查找待提取的节点标签：mxml_node_t *mxmlFindElement(mxml_node_t *node, mxml_node_t *top,const char *name, const char *attr,const char *value, int descend);
  4. 获取标签属性：const char *mxmlElementGetAttr(mxml_node_t *node, const char *name);
  5. 获取标签的文本内容： const char *mxmlGetText(mxml_node_t *node, int *whitespace);
  6. 释放节点，关闭文件
    示例：
    解析的文本：
1<?xml version="1.0" encoding="utf-8"?>
2
3<books>
4  <book id="001">
5    <author>wanger</author>
6  </book>
7</books>

代码：

 1#include"mxml.h"
 2#include<stdio.h>
 3
 4int main()
 5{
 6    FILE* fp = fopen("book.xml","r");
 7    // jiazai xml
 8    mxml_node_t* xml = mxmlLoadFile(NULL,fp,MXML_NO_CALLBACK);
 9    mxml_node_t* book = NULL;
10    mxml_node_t* author = NULL;
11    // find note
12    book = mxmlFindElement(xml,xml,"book","id",NULL,MXML_DESCEND);
13        //get attr
14    author = mxmlFindElement(book,xml,"author",NULL,NULL,MXML_DESCEND);
15    if(author == NULL)
16    {
17        printf("author error\n");
18    }
19    else
20    {
21        printf("book id is:%s\n",mxmlElementGetAttr(book,"id"));
22        printf("author is:%s\n",mxmlGetText(author,NULL));
23        book = mxmlFindElement(xml,xml,"book","id",NULL,MXML_DESCEND);
24    }
25    mxmlDelete(xml);
26    fclose(fp);
27    return 0;
28}

结果：

1book id is:001
2author is:wanger
