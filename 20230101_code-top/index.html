146.LRU缓存机制

关键字

关键字对应信息
键值对立马想到哈希表，而且可以通过O(1)时间通过键找到值
有出入顺序
(访问，插入数据的时间先后顺序)		首先想到 栈，队列 和 链表
更新数据顺序
(题目要求 get, put 后数据要被设置为最新数据)		1.说明要求可以随机访问
2.需要把数据插入到头部或者尾部
使用哈希表可以实现复杂度为O(1)的随机访问，如果哈希表的值包含链表的位置信息，那么就可以对链表进行O(1)随机访问。链表的顺序可以看做访问时间顺序，所以链表中的元素必须储存键，这样才能通过键找到哈希表的元素进而实现删除要求
